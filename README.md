My work covers **RAG systems, AI automation, full-stack development, SaaS products, APIs, and security utilities**. I ship production-ready systems on Laravel and FastAPI backends with React and Next.js frontends, backed by PostgreSQL, MySQL, and vector databases.

I work with founders, agencies, and SaaS teams on remote contract and project-based engagements, from short builds to long-term development partnerships.
